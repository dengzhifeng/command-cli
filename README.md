## 脚手架工具

### 功能和命令

-   自动提交代码 `command-cli push -m feat:修改`
-   替换文件 `command-cli replace-file --oldFile test-oldFile.js --newFile test-newFile.js`
-   合并代码

## 安装教程

-   全局安装 npm i command-cli -g

## 开发流程

-   yarn dev 开发模式
-   yarn build 打包模式（使用 rollup 打包 commonjs 格式）

## 查看帮助

command-cli --help
